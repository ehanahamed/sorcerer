# Custom layouts

Just like most other code editors, Sorcerer has a variety of possible layouts.

This is one possible layout,
with both outer areas visible
and four parts in the inner area
```
|-------|----------|----------|-------|
|       |          |          |       |
|       |----------|----------|       |
|       |          |          |       |
|-------|----------|----------|-------|
```

The inner area has a variety of possible layouts
```
|-------|---------------------|-------|
|       |                     |       |
|       |                     |       |
|       |                     |       |
|-------|---------------------|-------|

|-------|----------|----------|-------|
|       |          |          |       |
|       |          |          |       |
|       |          |          |       |
|-------|----------|----------|-------|

|-------|----------|----------|-------|
|       |          |          |       |
|       |          |----------|       |
|       |          |          |       |
|-------|----------|----------|-------|

|-------|----------|----------|-------|
|       |          |          |       |
|       |----------|          |       |
|       |          |          |       |
|-------|----------|----------|-------|

|-------|----------|----------|-------|
|       |          |          |       |
|       |----------|----------|       |
|       |          |          |       |
|-------|----------|----------|-------|
```

The outer areas can be hidden
```
|-------|--------------|--------------|
|       |              |              |
|       |--------------|--------------|
|       |              |              |
|-------|--------------|--------------|

|--------------|--------------|-------|
|              |              |       |
|--------------|--------------|       |
|              |              |       |
|--------------|--------------|-------|

|------------------|------------------|
|                  |                  |
|------------------|------------------|
|                  |                  |
|------------------|------------------|
```

This is how areas are named
```
|-------|----------|----------|-------|
|       |                     |       |
| outer |        inner        | outer |
|       |                     |       |
|-------|----------|----------|-------|

|-------|---------------------|-------|
|       |        upper        |       |
|       |---------------------|       |
|       |        lower        |       |
|-------|---------------------|-------|

|------------------|------------------|
|                  |                  |
|      left        |      right       |
|                  |                  |
|------------------|------------------|
```
